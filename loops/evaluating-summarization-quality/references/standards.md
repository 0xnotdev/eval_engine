## Dataset Provenance

- **Source**: Hand-authored original passages across six domains (news, legal, technical, narrative, scientific, business). Each passage is 3-8 sentences of fully written-out source text — not sourced from an external dataset.
- **License Assessed**: Yes — all passages are original work, Apache-2.0 (consistent with the repository license).
- **Method**: 20 passages were manually written with domain-diverse subject matter. Each row includes a complete source passage, a concrete reference summary, and a passage-specific rubric listing required facts and hallucination boundaries. Domains covered: space/aerospace, law, cybersecurity, sports, climate science, cloud computing, archaeology, finance/earnings, materials science, monetary policy, marine biology, AI regulation, gastronomy, fintech, urban transport, animation industry, education research, and space launch operations.
